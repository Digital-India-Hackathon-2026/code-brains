AI-powered Anti Money Laundering & Fraud Detection Platform

---

## Team Code Brains

Digital India Hackathon 2026

## Problem Statement

Financial institutions process millions of transactions daily, making it difficult to identify fraudulent or money laundering activities in real time.

MuleGuard leverages Machine Learning, Spring Boot, React, and FastAPI to detect suspicious transactions and generate risk insights.

---

## Features

- AI Fraud Detection
- Transaction Risk Scoring
- Money Mule Detection
- SAR Report Generation
- Interactive Dashboard
- CSV Upload
- Secure Authentication
- REST APIs
- Spring Boot Backend
- FastAPI ML Service

## Installation

### Clone

```bash
git clone https://github.com/Digital-India-Hackathon-2026/code-brains.git
```

---

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Runs on

```
http://localhost:5173
```

---

### Backend

```bash
cd backend-spring
./mvnw spring-boot:run
```

Runs on

```
http://localhost:8080
```

---

### ML Service

```bash
cd ml-service
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Runs on

```
http://localhost:8000
```
